Impressora 3D

Página com ligação para a drive e os passos para aceder à impressora 3D.

Ligação para Drive

Aponta a câmara do telemóvel
https://drive.google.com/drive/folders/1ct21WrbKPZxjoyv4Arwp_BGNeb_JJuAw

Nota: É necessário fazer login com uma conta Google para fazer upload dos ficheiros. Contudo, não é necessário fazer login para fazer download dos ficheiros.

Passos

  1. Abrir o ficheiro printer.sh
    O ficheiro printer.sh encontra-se no ambiente de trabalho.
  2. Usar uma pen ou a drive
    Transfere os ficheiros para um dispositivo de armazenamento ou faz upload para o Google Drive.
  3. Fazer upload do modelo para o Ultimaker
    Carrega o ficheiro do modelo no Ultimaker.
  4. Ligar a impressora
    O botão encontra-se no lado esquerdo da impressora.
